## Working Rules

- Prefer the nearest local `CLAUDE.md` for implementation details.
- Preserve established financial dashboard and reporting structure unless the task explicitly asks for redesign.
- Use local briefs and `brand_assets/` first.
- Only pull in shared Turnkey branding when the project explicitly needs Turnkey brand treatment.
